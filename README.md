# Alpha Sigma Phi Rho Colony Website

Need to install  [https://www.mongodb.org/](mongodb) seperately

Steps for running Locally: 

1. Clone
2. `npm install`
3. Start mongo - `sudo mongod`
4. `npm start`
5. [http://localhost:3000](http://localhost:3000)
